Typography

Suggested main font

The font used in the docs, examples and in this cocumentations is theThe Inter typeface familyNOT included in the CSS package

To include the font inter Inter just add

  @import url('https://rsms.me/inter/inter.css');
  :root {
    --font-family-sans-serif-custom: 'Inter';
  }
  @supports (font-variation-settings: normal) {
    :root {
      --font-family-sans-serif-custom: 'Inter';
    }
  }

Check the section `Add new font`

Add new font

to add a new font in the top of the stack, load the font, from Google fonts for exaple, and add the name of the font in top of font stack with the CSS ustom property:

:root { --font-family-sans-serif-custom: Name Font; }

its important to NOT add the default sugestion from Google fonts, with the `Name Font, sans-serif`, because the `sans-serif` will have higher priority than the rest of the default font stack.

Container

the grids or any other element can be nested inside a `.grid_container` class

the widths of the container per breakpoint:

  2. mobile_m: 100%
  3. mobile_l: 100%

  5. laptop: 992px
