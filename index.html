Font Family

The default font stack is:

'HelveticaNeue-Light', 'Helvetica Neue Light', 'Helvetica Neue', Helvetica, Arial, 'Lucida Grande', sans-serif

You can change the font adding a loading a new font and setting the font name in CSS. Example

Add Montserrat font from google fonts

<link href="https://fonts.googleapis.com/css?family=Montserrat&display=swap" rel="stylesheet">

And set the font name

:root { --font-family: Montserrat; }
