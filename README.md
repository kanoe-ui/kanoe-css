## Development

### CSS / Styles

Styles are builded with [stylus](http://stylus-lang.com/).

```sh
$  yarn stylus:build
$  yarn stylus:watch
```

Commits messages follow the [conventional commit format](https://conventionalcommits.org) and are checked with [commitlint](https://commitlint.js.org/).
